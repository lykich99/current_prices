== Description ==

This plugin can you help correct number the data which depend on the ratio.
number * coefficient = result.
As examples you sell products(Euro) what had buy in China(Yuan) and prices product depend on rate. 
The exchange rate is constantly changing. It may be important for post-Soviet space when the exchange rate not stabilized.
For support visit the [Support Forum](https://lweb.pl.ua/wordpress-plugin-correct_prices#support "Support Forum")

== Installation ==

1. Upload 'plugin-directory' to the '/wp-content/plugin/correct_prices/' directory
2. Activate the plugin through the 'Plugins' SubPanel in WordPress
3. Click for link setting.
4. Set the coefficient (for default value is 10).
5. Use TinyMCE Buttons "CorrectPricesCode" => "ShortCodeInsert" add shortcode on page.
   Add data use Buttosn "CorrectPricesCode" => "InsertDataCorrectPrices". You can add much numbers for correcting.
6. Save you post and see result.
